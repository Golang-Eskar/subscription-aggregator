# subscription-aggregator
Subscription Aggregator — REST-сервис, позволяющий хранить и управлять данными о пользовательских онлайн-подписках, а также агрегировать их стоимость за выбранный период. Сервис реализован на Go, использует PostgreSQL как хранилище и предоставляет HTTP API с документацией в формате Swagger. Запуск выполняется через Docker Compose.
